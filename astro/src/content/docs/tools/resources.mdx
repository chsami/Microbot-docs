Microbot offers a range of resources to support developers in creating, testing, and deploying custom scripts.
This guide provides a comprehensive list of materials and tools available to help you succeed in your scripting journey, whether you’re a beginner or an advanced developer.

<Card title="Maintaining Your Microbot Setup" icon="approve-check-circle">

</Card>

## Additional Learning Resources
### Recommended Tools and Setup Guides

<Steps>
1. **Maven Setup**: Use our [Maven Build Guide](/tools/maven) to set up your build environment, manage dependencies, and streamline project builds.

2. **IDE Configuration**: Follow setup guides for popular IDEs like [VSCode](/tools/vscode) and [IntelliJ](/tools/intellij) to optimize your scripting workspace.

3. **Git and Version Control**: Learn to use [Git and GitHub](/tools/git) for version control, enabling collaboration, tracking, and code management.
</Steps>
